In CSS Grid layout, grid-auto-row and grid-auto-column are properties that define the size of implicitly created rows and columns when grid items are placed outside the explicitly defined grid using the grid-row and grid-column properties.
grid-auto-row: Specifies the size of implicitly created rows.
grid-auto-column: Specifies the size of implicitly created columns.

Here the first row is 50 px as defined in css but the other two row are 100px because it has no defined row width so they are taking the row width specify by grid-auto-column
